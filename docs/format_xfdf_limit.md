# XFDFファイルフォーマット(制限)

PDFビューアやバージョンによって制限がある。

## 重大な制限

PDFビューアが正常に扱えない制限として、以下がある。

* XMLファイル形式でない。  
  PDFビューアは、読み込みできない。
* 埋め込みフォントは使用出来ない。  
  Edge で「コメントを追加」した場合、埋め込みフォントが使用される。  
  この場合、Acrobat Reader DC で開くとエラー表示がされる。  
  注釈は FDF 形式では保存できるが、 XFDF 形式では保存できない。

## 軽微な制限

PDFビューアの表示の違いとして、以下がある。

* Edge で表示する場合の制限
  * ハイライトの注釈テキスト、ノート注釈、およびテキストボックス以外は、日本語は文字化けする。
  * 注釈テキストのリッチテキストは正しく表示しない
  * ポップアップは、無視する
  * サウンドデータは、アイコン表示のみ
